# Colabs

Repositorio de mis laboratorios de Google Colab de la Universidad.

## Estructura

Este repositorio contiene notebooks de Google Colab organizados por asignatura o tema.

## Uso

1. Clona este repositorio
2. Abre los notebooks (.ipynb) en Google Colab
3. Ejecuta las celdas según sea necesario

## Organización

Los notebooks están organizados en carpetas por tema o asignatura para facilitar su localización.
